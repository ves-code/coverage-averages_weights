      49             : /*
      50             : Functions that measure whether values are less than a certain quantity.
      51             : 
      52             : Switching functions \f$s(r)\f$ take a minimum of one input parameter \f$d_0\f$.
      53             : For \f$r \le d_0 \quad s(r)=1.0\f$ while for \f$r > d_0\f$ the function decays smoothly to 0.
      54             : The various switching functions available in plumed differ in terms of how this decay is performed.
      55             : 
      56             : Where there is an accepted convention in the literature (e.g. \ref COORDINATION) on the form of the
      57             : switching function we use the convention as the default.  However, the flexibility to use different
      58             : switching functions is always present generally through a single keyword. This keyword generally
      59             : takes an input with the following form:
      60             : 
      61             : \verbatim
      62             : KEYWORD={TYPE <list of parameters>}
      63             : \endverbatim
      64             : 
      65             : The following table contains a list of the various switching functions that are available in plumed 2
      66             : together with an example input.
      67             : 
      68             : <table align=center frame=void width=95%% cellpadding=5%%>
      69             : <tr>
      70             : <td> TYPE </td> <td> FUNCTION </td> <td> EXAMPLE INPUT </td> <td> DEFAULT PARAMETERS </td>
      71             : </tr> <tr> <td>RATIONAL </td> <td>
      72             : \f$
      73             : s(r)=\frac{ 1 - \left(\frac{ r - d_0 }{ r_0 }\right)^{n} }{ 1 - \left(\frac{ r - d_0 }{ r_0 }\right)^{m} }
      74             : \f$
      75             : </td> <td>
      76             : {RATIONAL R_0=\f$r_0\f$ D_0=\f$d_0\f$ NN=\f$n\f$ MM=\f$m\f$}
      77             : </td> <td> \f$d_0=0.0\f$, \f$n=6\f$, \f$m=2n\f$ </td>
      78             : </tr> <tr>
      79             : <td> EXP </td> <td>

      85             : </td> <td> \f$d_0=0.0\f$ </td>
      86             : </tr> <tr>

      91             : </td> <td>

     140             : \verbatim

     142             : \endverbatim

     156             : keyword D_MAX to assert that for \f$r>d_{\textrm{max}}\f$ the switching function can be assumed equal to zero.
     157             : In this case the function is brought smoothly to zero by stretching and shifting it.
     158             : \verbatim
     159             : KEYWORD={RATIONAL R_0=1 D_MAX=3}
     160             : \endverbatim
     161             : the resulting switching function will be
     162             : \f$
     163             : s(r) = \frac{s'(r)-s'(d_{max})}{s'(0)-s'(d_{max})}
     164             : \f$

     166             : \f$
     167             : s'(r)=\frac{1-r^6}{1-r^{12}}
     168             : \f$
     169             : Since PLUMED 2.2 this is the default. The old behavior (no stretching) can be obtained with the
